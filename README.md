# ele-react

## 声明

很目的明确的说这是一个为了找工作而部署的一个react项目。原始目的是为了使用制作一个饿了么的h5客户端。但最后做成什么样谁知道呢。

## 部署步骤
1. `git clone git@github.com:BubblyFace/ele-react.git`
2. `cd ele-react`
3. `npm install`
4. `npm start` 热更新模式启动
    `npm run build` 部署模式
    `npm run watch` 观察者模式

## 使用技术栈

* 框架：webpack+react+react-router+react-redux
* UI：MaterialUI
